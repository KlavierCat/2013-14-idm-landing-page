TZD

An Interactive Narrative

TZD is a web-app that functions as an interactive narrative of Trinity College Dublin. The app riffs upon the currently fashionable 'zombie apocalypse' story trope and makes it the lens through which the user 'reflects' on Trinity. The user navigates the narrative through an interactive map and can view the narrative in chronological order on the inventory page.

  • Members: Panpan Lin, Fergus Kenny, Kyle Murphy, Ben Hardy, Ryan Yates
  • Supervisor: Marguerite Barry

SofaSesh

An Interactive Installation

SofaSesh is an interactive installation based upon a sitting room context. Every object within this setting has been installed with different types of sensors which are controlled by user behaviour. The result of user interaction is an immersive experience combining sound, lights and graphics.

  • Members: Aishling Costello,Conor McGee,Emily McVey,Evan Greally
  • Supervisor: Prof Declan O’Sullivan

TCD in Numbers

Data Visualisation

TCD in Numbers is an interactive data visualisation project providing users with a tool to extract meaning from typically inaccessible statistics pertaining to the student body of Trinity College Dublin. Through the medium of a website, users can access graphic representations of data spanning the last hundred years. Drawing on official records of Trinity College and the Higher Education Authority and focusing on the four major themes of Gender, Domiciliary, Faculty and Age, TCD in numbers reveals patterns and raises questions about data that would otherwise be hidden in banal statistics.

  • Members: Eoin Kilbride, Clíona de Paor and Jeremiah Ambrose
  • Supervisor: Marguerite Barry

Application

A 3D Game

Over the summer of 2014, IDM Team 4 of the M.Sc. in Interactive Digital Media at Trinity College Dublin developed and produced Application, a 3D game set in a surreal, brutalist world inspired by the TCD campus that satirizes the impedimentary role of bureaucracy in the academic world. The game was designed to be unwinnable and unending, in reflection of the frustrations of bureaucracy itself.

You can download the game or learn more about it here.

  • Members: Cormac Neeson, Donal Deery, Katie Canning
  • Supervisor: Marcin Gorzel
